# YieldVitals
YieldVitals Pro 專為旗艦級硬體與現代前端架構打造的網頁效能基準測試工具 (Benchmark)。為打破傳統跑分軟體在單執行緒與 Vsync 鎖幀上的盲區，本工具採用多核心 Web Worker 叢集進行高強度 CPU 浮點與矩陣壓測，並導入 Three.js 引擎執行十萬級 3D 實體動態光影渲染，真實榨乾 M-Series 或 RTX 級別的顯示卡算力。此外，更整合了嚴苛的 RAM 垃圾回收 (GC)、DOM 強制佈局重繪 (Layout Thrashing) 壓力測試，以及基於全球節點的 Gigabit 千兆並發網路吞吐評估。透過專屬的校準計分演算法與視覺化硬體天梯圖，YieldVitals Pro 能為開發者、QA 品管及硬體玩家提供最嚴苛、精準的設備極限檢驗報告。
